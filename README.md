# 🚀 API Centralizada - Backend Multiuso

Backend centralizado e reutilizável desenvolvido em **MySQL** para alimentar de forma integrada todo o meu ecossistema web atual e futuro.

---

## 📌 Objetivo do Projeto

Centralizar a regra de negócio e o armazenamento de dados em uma única API robusta para evitar a duplicação de bancos de dados. Com essa arquitetura escalável, os dados são consumidos dinamicamente por diferentes aplicações.

### 🌐 Aplicações Atendidas:
* **Currículo Web:** Alimentação dinâmica de experiências, habilidades e dados profissionais.
* **Portfólio Moderno:** Gerenciamento centralizado de projetos, tecnologias e links.
* **Landing Page de Vendas (Futuro):** Estrutura preparada para receber o sistema focado em conversão e captação de leads.

---

## 📐 Benefícios da Arquitetura

* **Total Reutilização:** Uma única base de dados MySQL servindo múltiplos frontends de forma independente.
* **Fácil Escalabilidade:** Pronta para receber novas tabelas e rotas para novos projetos sem quebrar os sites existentes.
* **Consistência:** Alterações feitas no banco refletem instantaneamente em todas as pontas conectadas.
